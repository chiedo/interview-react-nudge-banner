# Interview starter — NudgeBanner

This is a small React + TypeScript starter for a 30-minute live coding interview.

**🔗 Live preview of the starting state: <https://chiedo.github.io/interview-react-nudge-banner/>**

## Open this in a browser sandbox (no install needed)

The fastest path — works with **no signup**:

**👉 [Open in StackBlitz](https://stackblitz.com/github/chiedo/interview-react-nudge-banner)**

Other options:

- **CodeSandbox**: paste the repo URL into <https://codesandbox.io/p/import>.
- **GitHub Codespace**: fork this repo, then click **Code → Codespaces → Create**.
- **Locally**: clone, `npm install`, `npm run dev`, `npm test`.

You may use any AI assistant you like (Copilot, ChatGPT, Claude, etc.). Bring whatever you use day to day — your interviewer wants to see how you drive it.

## What to do once it's open

1. Open **`TASK.md`** — the prompt for the interview.
2. Open **`MOCK.md`** — an annotated mock of the visual target.
3. Read **`src/NudgeBanner.tsx`** and **`src/__tests__/NudgeBanner.test.tsx`** before you start coding.
4. The seed test should already pass: run `npm test` to confirm.
5. Then extend the component to satisfy the requirements in `TASK.md`.

You don't need to be fast — your interviewer is watching how you think, prompt, and validate. Talk out loud.

## Scripts

```bash
npm install        # install deps (StackBlitz does this for you)
npm run dev        # start Vite dev server
npm test           # run vitest once
npm run test:watch # run vitest in watch mode
npm run build      # type-check + production build
```
